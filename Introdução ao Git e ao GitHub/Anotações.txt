# Sobre o Git

<p>O Git permite que você revise como seu projeto cresce e facilmente olhe ou restaure estados de arquivo do passado. Uma vez conectado a uma rede, o Git permite que você envie seu projeto para o GitHub ou outras alternativas, como: Bitbucket, Beanstalk ou GitLab para compartilhar e colaborar com outros desenvolvedores.</p>
